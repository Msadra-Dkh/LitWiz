# ✨ Features

- ⚡ Instantly convert mistyped text
- 🌐 Automatically switch keyboard layout after conversion
- ⌨️ Global hotkeys(Changable)
- 🖥️ Runs quietly in the system tray
- 💡 Lightweight and fast
- 🎭 Have notification & sound

# 👾 How It Works

1. Select the mistyped text.
2. Press your LitWiz hotkey(It is already F7).
3. LitWiz converts the text instantly.
4. Your keyboard layout switches automatically.

---

# 📋 Requirements

- Windows 10 / 11
- No installation required

---

# 🚀 Getting Started

1. Download the latest release.
2. Extract the ZIP file.
3. Run `LitWiz.exe`.
4. Select any Persian or English text.
5. Press **F7**.

---

# ⌨️ Default Hotkeys

| Hotkey | Action |
|---------|--------|
| F7 | Convert selected text |

**You can choose any hotkey you like!**👨‍💻

**just be careful not to use reserved system shortcuts.**🚨
